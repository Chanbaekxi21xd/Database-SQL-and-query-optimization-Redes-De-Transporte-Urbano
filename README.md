# Database-SQL-and-query-optimization-Redes-De-Transporte-Urbano
Diseño y análisis de una base de datos relacional para redes de transporte urbano, aplicando normalización hasta Tercera Forma Normal (3FN). Desarrollo de scripts SQL para creación de tablas, carga de datos y consultas analíticas, con el fin de obtener información útil para la gestión y optimización del sistema de transporte.
